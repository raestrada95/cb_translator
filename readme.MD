**cb_translator** is a command-line tool that allows you to translate subtitles files in srt using azure 


Installation
------------
cb_srtTranslator::

    $ pip install cb_translator


Usage
-----
CLI
^^^
Translate from srt to Spanish::

    $ cbsrt -i file.srt -l es
